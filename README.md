## 작성 규칙

- 폴더명과 파일명을 가능하면 영문으로 작성한다.
- 해당 문서를 처음 보는 사람도 필요한 내용을 찾기 쉽도록 분류한다.
- 한번 작성한 문서는 다시 검색하는 추가비용이 들지 않도록 자세히 작성한다.
- 느린것보다 멈추는 것이 무서운 것이므로 느리더라도 꾸준히 기록하는것을 목표로 한다.
